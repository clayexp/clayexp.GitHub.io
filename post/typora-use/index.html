Typora使用教程

2021-09-11 / 1 min read

Typora是一款轻便简洁的Markdown编辑器，支持即时渲染技术，这也是与其他Markdown编辑器最显著的区别。即时渲染使得你写Markdown就想是写Word文档一样流畅自如，不像其他编辑器的有编辑栏和显示栏。 优点:
简洁美观
实时预览
扩展语法
跨平台
免费
简单使用

  1. 大标题：通过ctrl + 数字 1 2 3 ....方式，还可以通过加# 的方式

  2. 序号标题：

    1. 1号
    2. 2号
    3. 3号
      1. 按一个tab键 就是这个。

  3. 列

    • 圆圈序号 通过一个 + 空格 形成了。
      • 正方实心 通过一个tab键 就可以。

  4. 插入代码块

    a = 1
b = 2
c = 3

    三个 ``` + python回车就可以。

    或者 ctrl + shift + k

    a = 1
b = 2
c = 3
